# Cineteca

## Nom dels integrants  
**Araceli Pacheco**  

## Nom del projecte  
**Cineteca**  

## Descripció  
Cineteca és una plataforma web per a la gestió de sessions de cinema, permetent als usuaris consultar cartelleres, comprar entrades i gestionar reserves.
